# План реализации карты для Satellite Tracker

## Текущий статус реализации

| Требование | Статус | Примечания |
|-----------|--------|------------|
| 2D map (dark CartoDB tile) | ✅ Готово | Используется MapLibre GL JS (dark-matter-gl-style) |
| 3D map view (CesiumJS) | ❌ Не реализовано | Бонус |
| Real-time satellite positions | ✅ Готово | Маркеры обновляются каждую секунду |
| Animated satellite movement | ✅ Готово | Позиции обновляются через Web Worker |
| Standard map controls | ✅ Готово | NavigationControl (zoom/pan) |
| 3D orbital space view | ❌ Не реализовано | Бонус |

---

## Реализованные функции

### 1. 2D карта с темной темой
- **Библиотека**: MapLibre GL JS + react-map-gl
- **Стиль**: CartoDB Dark Matter GL
- **URL**: `https://basemaps.cartocdn.com/gl/dark-matter-gl-style/style.json`

### 2. Отображение спутников
- **Цветовое кодирование по типу орбиты**:
  - LEO: `#00ff88` (зеленый)
  - MEO: `#ffaa00` (оранжевый)
  - GEO: `#ff4466` (красный)
  - HEO: `#aa88ff` (фиолетовый)
- **Обновление позиций**: каждую 1 секунду
- **Выбор спутника**: клик по маркеру

### 3. Визуальное выделение выбранного спутника
- Белое кольцо вокруг выбранного спутника
- Увеличенный размер маркера
- Автоматическое центрирование на спутнике (flyTo)

### 4. Управление картой
- Зум (кнопки + / -)
- Панорамирование (drag)
- Вращение (Shift + drag)

### 5. Дополнительные функции
- **GroundTrack**: отображение орбитального следа
- **CoverageCone**: отображение зоны видимости
- **Фильтры**: по типу орбиты, назначению, названию

- `react-map-gl`: ^8.1.0
